# Hello-Jest-TS

「Jestではじめるテスト入門」のサンプルプロジェクトです。書籍で利用されているサンプルコードがこのプロジェクトに含まれています。 

03_getting_stated_jest: 2.2 セットアップとはじめてのテスト
04_assertion: 2.3 テストの結果と評価（Matcher マッチャー）
06_group : 2.4 テストのグループ化と前後処理
07_mock : 2.6 モックを利用したテスト
08_ui : 2.7 UIテスト
09_e2e : 2.8 E2Eテスト

## パッケージのインストール
```
$ npm install
```

## テストの実行
```
$ npm run test 
```
